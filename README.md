# Wirtschaft & Märkte

Statische GitHub-Pages-Seite zur gekürzten Zusammenfassung von `Ökonomisches Handeln auf Märkten`.

## Inhalt

- Scrollbare Themenübersicht
- Dropdown-Navigation mit Sprung zum Abschnitt
- Mobile-optimiertes Einzeldatei-Frontend

## Deployment

Die Seite läuft direkt über GitHub Pages aus dem Root der `main`-Branch.
